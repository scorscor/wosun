# 莴笋录屏

一款简洁高效的桌面录屏软件，支持高清录制、麦克风录音，生成iPhone兼容的MP4视频。

## 功能特性

### 核心功能
- **高清录制**：支持最高4K分辨率（3840x2160）屏幕录制
- **麦克风录音**：同步录制麦克风语音，支持实时音量测试
- **智能倒计时**：3秒倒计时，录制准备更从容
- **浮窗指示器**：录制时显示小型浮窗（Logo + 计时器），不遮挡内容
- **自由拖动**：浮窗可拖动到屏幕任意位置，自动吸附到最近边框
- **悬停控制**：鼠标悬停浮窗展开停止按钮，操作便捷

### 视频质量
- **多档清晰度**：标清(2.5M) / 高清(8M) / 超清(12M) / 极清(16M)
- **标准帧率**：30fps流畅录制
- **iPhone兼容**：生成标准H.264编码MP4，支持所有iPhone/iPad设备
- **快速转换**：自动转换为标准MP4格式，兼容所有主流播放器

### 便捷功能
- **自定义文件名**：保存时可修改文件名
- **自定义保存目录**：可设置录屏文件保存位置
- **一键打开文件夹**：保存后点击文件夹图标快速定位文件
- **视频预览**：保存后直接在应用内预览视频
- **调试模式**：开发者可开启调试模式查看详细日志

## 使用方法

### 开始录制

1. 启动应用，点击 **开始录制** 按钮
2. 3秒倒计时后自动开始录制
3. 主窗口隐藏，屏幕边缘出现莴笋浮窗
4. 浮窗实时显示录制时长

### 停止录制

1. 鼠标移动到浮窗位置
2. 浮窗自动展开，显示停止按钮
3. 点击 **停止** 结束录制
4. 弹出保存对话框，可修改文件名
5. 点击 **保存** 开始转换视频
6. 转换完成后显示保存路径和视频预览

### 浮窗操作

- **拖动**：按住浮窗拖动到任意位置
- **吸附**：松手后自动吸附到最近的屏幕边框
- **展开**：鼠标悬停1.5秒后展开停止按钮
- **收起**：鼠标移开后自动收起

## 菜单功能

### 配置菜单

- **保存到...**：设置录屏文件保存目录
- **调试模式**：开启/关闭开发者工具
- **清晰度**：选择视频质量（标清/高清/超清/极清）
- **测试麦克风**：打开麦克风音量测试窗口

### 帮助菜单

- **说明**：查看本使用说明
- **关于**：查看软件版本和作者信息

## 麦克风测试

在配置菜单中点击 **测试麦克风**，可以：
- 实时查看麦克风音量百分比
- 查看音量进度条
- 查看40段频谱可视化效果

确保麦克风正常工作后再开始录制。

## 视频格式说明

生成的MP4视频采用以下标准：
- **视频编码**：H.264 Baseline Profile（最兼容）
- **音频编码**：AAC-LC 128kbps
- **分辨率**：根据屏幕自动适配（最高4K）
- **帧率**：30fps
- **兼容性**：支持iPhone、iPad、Windows、Mac、Android所有设备

## 常见问题

**Q: 为什么转换需要一些时间？**
A: 为了确保视频兼容所有设备（特别是iPhone），需要将录制的WebM格式转换为标准MP4格式。转换速度取决于视频长度和电脑性能。

**Q: 如何提高录制质量？**
A: 在配置菜单中选择更高的清晰度档位（超清或极清）。

**Q: 视频文件保存在哪里？**
A: 默认保存在系统视频文件夹，可在配置菜单中修改保存位置。保存后点击文件夹图标可快速打开文件所在位置。

**Q: 为什么iPhone无法播放视频？**
A: 最新版本已优化为iPhone兼容格式，如仍有问题请确保使用最新版本。
